
The `list-bare` object simply removes bullets and indents from lists.


## Install using npm:

```ssh
    $ npm install --save-dev aleut.objects.list-bare

```

## Usage

The `list-bare` object just have one class:

```html
<ul class="o-list-bare">
	<li>List item<li>
	<li>List item</li>
</ul>
```
